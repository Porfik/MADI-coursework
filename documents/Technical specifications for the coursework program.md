# Техническое задание программы для курсовой

## Тема курсовой работы:

>Разработка программного приложения для расчёта минимально необходимого расстояния видимости для безопасного движения транспортных средств на дороге при изменяемых значениях заданных параметров с выводом результатов в виде таблицы и графиков.

## Что требуется реализовать в UI программы:

1. Заголовок окна: содержит название приложения, а также информацию о названии и положении проекта и его статусе (сохранён или нет).
2. Вкладки:
    1. "Файл" (подменю содержащее пункты для работы с файлами, за исключением пункта "Выход"):
        1. "Создать" (сбрасывает все значения и результаты вычислений, отвязывает уже имеющиеся сохранённые файлы);
        2. "Открыть..." (открытие файла с исходными данными);
        3. "Сохранить ввод" (сохранение исходных данных в уже имеющийся файл);
        4. "Сохранить ввод как..." (сохранение исходных данных в новый файл);
        5. "Сохранить результаты" (сохранение результатов вычислений в уже имеющийся файл);
        6. "Сохранить результаты как..." (сохранение результатов вычислений в новый файл);
        7. "Выход" (закрыть приложение, отделяется от остальных пунктов горизонтальной чертой).
    2. "Результаты" (подменю содержащее пункты для работы с результатами):
	    1. "Вычислить" (дублирует кнопку для расчёта результатов).
3. Секция ввода параметров:
    1. Поля ввода данных:
        1. Время реакции водителя (t). Реализуется через поле текстового ввода, с возможностью последовательного ввода нескольких вариантов переменной через ";" (значения по умолчанию: "1,5; 2");
        2. Расчётная скорость движения (V). Включает в себя три элемента:
            1. Первое поле текстового ввода, в котором указывается минимальная скорость (значение по умолчанию: "20");
            2. Второе поле текстового ввода, в котором указывается максимальная скорость (значение по умолчанию: "60");
            3. Ползунок с помощью которого выбирается значение шага изменения скорости. Динамически подстраивает своё максимальное значение в зависимости от минимальной и максимальной скорости (выставленное значение по умолчанию: "5").
        3. Коэффициент сцепления колёс автомобиля с дорогой (φ). Реализуется через поле текстового ввода, с возможностью последовательного ввода нескольких вариантов переменной через ";" (значения по умолчанию: "0,2; 0,3; 0,4");
        4. Продольный уклон (i). Реализуется через поле текстового ввода, с возможностью последовательного ввода нескольких вариантов переменной через ";" (значение по умолчанию: "0");
        5. Зазор безопасности (l1). Реализуется через поле текстового ввода, с возможностью последовательного ввода нескольких вариантов переменной через ";" (значение по умолчанию: "3").
    2. Функциональные кнопки:
        1. Кнопка "Рассчитать". При нажатии проверяет корректность введённых данных, если обнаружена ошибка то выводится соответствующее сообщение, иначе данные передаются в обработку.

    2. Графики.

## Уточнения от преподавателя:

1. Знак в формуле - минус, по сути все знаки инвертируются;
2. Как реализовать ввод и какие данные можно изменять - все данные изменяемы, ввод реализовать по своему усмотрению;
3. В чем задаётся продольный угол - в процентах;
4. Нужен ли функционал открытия сохранённых данных - нет.
